Module dom

source code

Peach 2 DOM. Peach XML files are parsed into an object model using these classes. The document object is Peach and all objects extend from Element.

Elements used inside of <DataModel>'s are called data elements and extend from DataElement. This also allows easy checking for a data element vs. a relation or other object by saying isinstance(obj, DataElement).

For speed, a native deepcopy was implemented in cDeepCopy, also some of the tree walking code for locating objects by name is done in native code for performance reasons (see cPeach).

API documentation is generated using epydoc and the script tools\gendocs.bat. Output ends up in docspidocs\index.html.
