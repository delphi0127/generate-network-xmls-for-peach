Class StaticBinary

source code

Contains some binary data. Can be set by string containing several formats of binary data such as " FF FF FF FF " or "ÿ ÿ ÿ", etc.

Example:

>>> gen = StaticBinary("""41 41 41
... 41 41 41 41 0x41 0x41""")
>>>
>>> print gen.getValue()
AAAAAAAAA
